Quiz Application

  • Category: React Project

This is my quiz app, that asks 3 different questionsg.

For this project, I aimed to create a simple quiz app using React Native, focusing on handling different types of questions (multiple-choice, multiple-answer, true/false) and navigating between questions seamlessly. The project allowed me to utilize React Native for mobile app development, along with navigation features from React Navigation. One major challenge I encountered was ensuring the correct rendering of the summary page, accurately displaying whether each question was answered correctly or incorrectly. To address this, I carefully reviewed the logic for evaluating answers and adjusted the rendering accordingly. In terms of design decisions, I chose a clean and minimalistic layout for the app, with contrasting colors for buttons and highlights to improve readability and user experience. The color scheme of orange, green, and blue was selected for its vibrancy and ability to attract attention without overwhelming the user.
